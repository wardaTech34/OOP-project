## Description
This is a **basic Object-Oriented Programming (OOP) project** that simulates a simple bank account system. Users can **create an account, deposit money, withdraw money, and check their balance**.  
The project demonstrates fundamental OOP concepts such as **classes, objects, encapsulation, and methods**.

## Features
- **Create Account:** Initialize a bank account with a name and starting balance.  
- **Deposit Money:** Add funds to the account.  
- **Withdraw Money:** Remove funds from the account, with balance validation.  
- **Check Balance:** Display the current account balance.  
- **OOP Concepts Demonstrated:** Class design, methods, object creation, and data encapsulation.

## How to Use
1. Run the program in a supported IDE or compiler (Java, C++, or Python depending on implementation).  
2. Follow prompts to create a bank account.  
3. Deposit, withdraw, or check balance using the program menu.  

## Purpose
This project helps beginners **understand the core principles of OOP** by simulating real-world bank transactions. It shows how to manage data securely within objects and how to structure code using classes and methods.
